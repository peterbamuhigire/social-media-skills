# World-Class Social Media Consultancy Engine Roadmap

**Repository:** `social-media-skills`  
**Date:** 2026-04-14  
**Planning basis:** repository README, `docs/evaluation/2026-04-13/*`, and the external analysis in `C:\Users\Peter\.claude\skills\docs\analysis`

## Purpose

This roadmap turns the current repository from a strong East Africa-first consultancy system into a world-class social media and digital marketing engine that can:

- diagnose markets and sectors before strategy is written,
- generate localised strategies and channel plans across countries,
- produce commercially credible proposals, plans, and evaluations,
- operate with stronger governance, quality control, and measurement discipline,
- use AI as a force multiplier without reducing trust or quality,
- support a consultancy model that can scale from boutique practice to premium advisory firm.

## Strategic Diagnosis

The core problem is not lack of content coverage. The repository already covers a large portion of the consultancy lifecycle. The critical weakness is that the system is still too dependent on Uganda and East Africa defaults. It needs a mandatory adaptation layer, stronger evaluation logic, more enterprise-grade commercial depth, and a clearer operating model for repeatable quality.

## End-State Definition

At the end of this roadmap, this repository should function as a world-class consultancy engine with these characteristics:

1. Every major deliverable starts with explicit market, sector, audience, risk, and regulatory context.
2. Strategy outputs make real choices about where to play, how to win, what not to do, and how success will be measured.
3. Platform, content, paid, and community recommendations change by market instead of inheriting one regional default.
4. Commercial documents can support boutique, SME, multinational, public sector, and regulated-client work.
5. Reporting and evaluation skills can connect activity to business outcomes, decision quality, and next actions.
6. AI is embedded as an efficiency and intelligence layer, with strong human review, cultural calibration, and governance.
7. The system is teachable, auditable, and usable by a growing consultancy team rather than only by a single expert operator.

## Operating Rule For The Whole Programme

Do not try to rewrite the entire repository at once. Build the missing system layer first, then retrofit the highest-leverage skills in waves. The principle is:

`context first -> strategy second -> execution third -> evaluation fourth -> scale fifth`

## Success Test

This roadmap succeeds when the repository can take a brief for a client in any viable market, produce a context-aware strategy and digital marketing plan, show how it would be executed, explain how it would be evaluated, and package that work as a premium consultancy offer without falling back on hidden East Africa assumptions.
