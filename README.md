# social-media-skills

A complete skill suite for running a professional social media and digital marketing consultancy in Uganda and East Africa. Covers every deliverable in the consultancy lifecycle ‚Äî from credentials and proposals through to strategy, content, campaigns, reporting, training, and AI-assisted production.

**Default context:** Uganda / East Africa ¬∑ British English ¬∑ UGX pricing ¬∑ EAT (UTC+3)

---

## How Skills Work

Each skill lives in its own directory containing a `SKILL.md` file. The frontmatter `description` field is the trigger ‚Äî Claude reads it to decide when to invoke the skill. Skills produce text documents: strategies, plans, playbooks, reports, slide outlines, and copy. No code, no graphic design, no video production.

## Key Conventions

- **British English throughout** ‚Äî organisation, colour, programme, behaviour, analyse
- **Uganda/East Africa default** ‚Äî UGX pricing, EAT timezone, WhatsApp-first, Mobile Money payments
- **500-line limit** per SKILL.md ‚Äî detailed references go in `references/` subdirectory
- **Every skill includes** Required Input and Quality Criteria sections
- **Deck skills** output slide-by-slide markdown: Headline / Bullets / Speaker Notes / Visual Direction
- **AI integration** ‚Äî every strategy and plan skill references how AI is used and how quality is controlled

## Strategic Frameworks Used

- POEM model ‚Äî Paid/Owned/Earned/Media (Chaffey, 2024)
- PESO model ‚Äî Paid/Earned/Shared/Owned (Dietrich, 2020)
- RACE framework ‚Äî Reach/Act/Convert/Engage (Chaffey, 2024)
- 10-4-1 rule ‚Äî Bodnar and Cohen (2012)
- Hero/Hub/Hygiene ‚Äî content tier model
- Like-Know-Trust (LKT) ‚Äî trust-building sequence
- SMART objectives ‚Äî all goals are Specific, Measurable, Achievable, Relevant, Time-bound
- ROI formula ‚Äî (TLV ‚àí COCA) √∑ COCA ‚Äî Bodnar and Cohen (2012)
- Consumer Value Journey (CVJ) ‚Äî DigitalMarketer / Ryan Deiss (8 stages)
- AI Marketing Canvas ‚Äî Venkatesan & Lecinski (2026) ‚Äî 5 steps √ó 4 customer moments
- Alpha-Beta-Gamma-Delta-Epsilon prompt structure ‚Äî Upadhyay (2024)
- STEPPS virality framework ‚Äî Berger (2013)
- Audacity Index ‚Äî Schaefer (2025)
